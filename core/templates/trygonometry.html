Тригонометрия

Определение

Тригонометрия — это раздел математики, изучающий отношения между сторонами и углами треугольников.

Основные тригонометрические функции

  • Синус: sin(θ) = противолежащий катет / гипотенуза
  • Косинус: cos(θ) = прилежащий катет / гипотенуза
  • Тангенс: tan(θ) = sin(θ) / cos(θ) = противолежащий катет / прилежащий катет
  • Котангенс: cot(θ) = cos(θ) / sin(θ) = прилежащий катет / противолежащий катет

Изображение: Определение синуса и косинуса

Основные тригонометрические тождества

  • Основное тригонометрическое тождество: sin²(θ) + cos²(θ) = 1
  • Формулы тангенса и котангенса:

    • tan(θ) = sin(θ) / cos(θ)

    cot(θ) = cos(θ) / sin(θ)

  • Формулы двойного угла:

    • sin(2θ) = 2sin(θ)cos(θ)

    cos(2θ) = cos²(θ) - sin²(θ)

    tan(2θ) = 2tan(θ) / (1 - tan²(θ))

Тригонометрия в прямоугольном треугольнике

В прямоугольном треугольнике для нахождения сторон и углов можно использовать тригонометрические функции.

Пример: если известны один катет и угол, можно найти другую сторону с помощью формул:

a = c * sin(θ)

b = c * cos(θ)

Применение тригонометрии

  • Астрономия и физика (расчет траекторий движения).
  • Геодезия и картография (измерение расстояний и высот).
  • Инженерия и архитектура (строительство и проектирование зданий).
  • Компьютерная графика и анимация.
