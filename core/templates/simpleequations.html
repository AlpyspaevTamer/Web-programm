Простейшие уравнения

Что такое уравнение?

Уравнение – это математическое выражение, в котором одна или несколько переменных связаны с числами и арифметическими операциями. Решением уравнения является значение переменной, при котором равенство становится верным.

Линейные уравнения

Линейное уравнение имеет вид: ax + b = 0, где a и b – числа, а x – переменная.

Пример решения уравнения:
2x + 3 = 7
2x = 7 - 3
2x = 4
x = 4 / 2 = 2.

Уравнения с неизвестным в знаменателе

Пример уравнения:
\( \frac{5}{x} = 2 \)
Решение:
Умножаем обе части на x:
5 = 2x
x = 5 / 2 = 2.5.

Квадратные уравнения

Квадратное уравнение имеет вид: ax² + bx + c = 0. Оно решается с помощью дискриминанта:

\( D = b^2 - 4ac \)

Если D > 0, уравнение имеет два корня:
\( x_1 = \frac{-b + \sqrt{D}}{2a} \), \( x_2 = \frac{-b - \sqrt{D}}{2a} \).

Если D = 0, уравнение имеет один корень:
\( x = \frac{-b}{2a} \).

Применение уравнений

Простейшие уравнения используются в физике, экономике, инженерии и повседневной жизни для решения задач, связанных с неизвестными величинами.
