Производная и её применение

Определение

Производная функции — это предел отношения приращения функции к приращению аргумента при стремлении последнего к нулю:

f'(x) = lim(Δx → 0) (f(x + Δx) - f(x)) / Δx

Геометрически производная выражает скорость изменения функции и равна угловому коэффициенту касательной к графику функции в данной точке.

Основные правила дифференцирования

  • Производная константы: (C)' = 0
  • Производная степени: (xⁿ)' = n * xⁿ⁻¹
  • Производная суммы: (f(x) + g(x))' = f'(x) + g'(x)
  • Производная произведения: (f(x)g(x))' = f'(x)g(x) + f(x)g'(x)
  • Производная частного: (f(x)/g(x))' = (f'(x)g(x) - f(x)g'(x)) / g²(x)
  • Производная сложной функции: (f(g(x)))' = f'(g(x)) * g'(x)

Производные основных функций

  • (sin x)' = cos x
  • (cos x)' = -sin x
  • (tan x)' = 1 / cos² x
  • (ln x)' = 1 / x
  • (e^x)' = e^x

Изображение: Касательная к графику функции

Применение производной

1. Исследование функции

Производная используется для нахождения критических точек функции:

- Если f'(x) > 0, функция возрастает.

- Если f'(x) < 0, функция убывает.

- Если f'(x) = 0, точка является стационарной и может быть максимумом или минимумом.

2. Нахождение экстремумов

Чтобы найти максимумы и минимумы функции, нужно:

  • Найти производную f'(x).
  • Решить уравнение f'(x) = 0.
  • Определить знак f''(x) во втором производном тесте (если f''(x) > 0, это минимум, если f''(x) < 0, это максимум).

3. Касательная и нормаль

Уравнение касательной к графику функции в точке (x₀, f(x₀)):

y = f'(x₀)(x - x₀) + f(x₀)

4. Применение в физике

  • Скорость движения: v = s'(t), где s(t) — координата в момент времени.
  • Ускорение: a = v'(t).
  • Максимальная и минимальная энергия в механике.

5. Оптимизация

Используется в экономике, инженерии и бизнесе для нахождения оптимальных значений (например, минимизация затрат, максимизация прибыли).
