Функции

Определение

Функция — это зависимость одной переменной (зависимой) от другой (независимой), при которой каждому значению независимой переменной соответствует единственное значение зависимой переменной.

Способы задания функции

  • Аналитический способ — с помощью формулы, например: y = 2x + 3.
  • Графический способ — с помощью графика на координатной плоскости.
  • Табличный способ — с помощью таблицы значений.
  • Словесный способ — с помощью описания зависимости.

Основные виды функций

  • Линейная функция: y = kx + b
  • Квадратичная функция: y = ax² + bx + c
  • Обратная пропорциональность: y = k/x
  • Показательная функция: y = a^x
  • Логарифмическая функция: y = loga(x)
  • Тригонометрические функции: y = sin(x), y = cos(x), y = tan(x)

Графики функций

График функции — это множество точек, координаты которых удовлетворяют заданному уравнению функции.

Примеры:

  • Линейная функция — прямая линия.
  • Квадратичная функция — парабола.
  • Обратная пропорциональность — гипербола.
  • Показательная функция — экспоненциальный график.
