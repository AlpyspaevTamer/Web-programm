Интегралы

Определение

Интеграл – это обобщение суммирования бесконечного количества бесконечно малых величин.

Различают два основных типа интегралов:

  • Неопределённый интеграл – это первообразная функции:

    • ∫ f(x) dx = F(x) + C

  • Определённый интеграл – это численное значение площади под графиком функции:

    • ∫[a,b] f(x) dx = F(b) - F(a)

Основные свойства интегралов

  • Линейность: ∫ (a·f(x) + b·g(x)) dx = a∫ f(x) dx + b∫ g(x) dx
  • Интеграл от суммы равен сумме интегралов: ∫ (f(x) + g(x)) dx = ∫ f(x) dx + ∫ g(x) dx
  • Интеграл от производной: ∫ f'(x) dx = f(x) + C
  • Первообразная постоянной: ∫ C dx = Cx + C

Таблица основных интегралов

  • ∫ xⁿ dx = (xⁿ⁺¹) / (n+1) + C, при n ≠ -1
  • ∫ e^x dx = e^x + C
  • ∫ sin x dx = -cos x + C
  • ∫ cos x dx = sin x + C
  • ∫ 1/x dx = ln|x| + C

Определённый интеграл как площадь под графиком функции

Методы интегрирования

  • Разложение на слагаемые: разбиение сложных функций на более простые.
  • Замена переменной: используется, когда можно заменить часть выражения на новую переменную.
  • Интегрирование по частям: ∫ u dv = uv - ∫ v du
  • Тригонометрические подстановки: применяются для выражений, содержащих корни и тригонометрические функции.

Применение интегралов

  • Нахождение площади фигур.
  • Определение длины дуги кривой.
  • Расчёт объёмов тел вращения.
  • Физические применения: вычисление работы, массы, центра тяжести.
  • Анализ динамических систем в механике и электротехнике.
