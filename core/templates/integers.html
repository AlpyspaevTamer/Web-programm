Целые числа и координаты

Что такое целые числа?

Целые числа – это числа без дробной части. Они включают:

  • Отрицательные числа (..., -3, -2, -1).
  • Ноль (0).
  • Положительные числа (1, 2, 3, ...).

Сравнение целых чисел

Число больше другого, если оно расположено правее на числовой оси. Например, -2 меньше 3, потому что находится левее на оси.

Координатная прямая

Числовая ось – это прямая, на которой числа расположены в порядке возрастания слева направо.

Координатная плоскость

Координаты используются для обозначения местоположения точек. Плоскость состоит из:

  • Оси X (горизонтальная).
  • Оси Y (вертикальная).

Каждая точка обозначается парой чисел (x, y). Например, (3, -2) означает точку с координатами 3 по оси X и -2 по оси Y.
