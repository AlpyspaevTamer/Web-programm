Теорема Пифагора

Формулировка

Теорема Пифагора утверждает, что в прямоугольном треугольнике квадрат гипотенузы равен сумме квадратов катетов.

Математическая запись:

c² = a² + b²

Где:

  • c — гипотенуза (самая длинная сторона);
  • a и b — катеты (две другие стороны).

Изображение: Прямоугольный треугольник

Примеры

Пример 1: Пусть a = 3, b = 4. Найдем гипотенузу:

c² = 3² + 4² = 9 + 16 = 25

c = √25 = 5

Пример 2: Пусть c = 13, b = 5. Найдем катет a:

a² = c² - b² = 13² - 5² = 169 - 25 = 144

a = √144 = 12

Практическое применение

  • В строительстве для проверки прямых углов.
  • В навигации и картографии для расчета расстояний.
  • В физике и инженерии для вычислений в механике и электротехнике.
