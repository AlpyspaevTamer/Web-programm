Логарифмы и показательные уравнения

Логарифмы

Логарифм числа b по основанию a – это показатель степени, в которую нужно возвести число a, чтобы получить b:

loga(b) = x, если ax = b

Пример: log2(8) = 3, так как 2³ = 8.

Свойства логарифмов

  • Произведение: loga(xy) = loga(x) + loga(y)
  • Деление: loga(x/y) = loga(x) - loga(y)
  • Степень: loga(xn) = n loga(x)
  • Логарифм основания: loga(a) = 1
  • Логарифм единицы: loga(1) = 0
  • Формула перехода к новому основанию: loga(b) = logc(b) / logc(a)

Показательные уравнения

Показательные уравнения – это уравнения, в которых переменная находится в показателе степени:

ax = b

Методы решения:

  • Приведение к одному основанию: Если возможно, выражаем оба числа через одно основание и сравниваем показатели.
  • Логарифмирование: Если выражения нельзя привести к одному основанию, берём логарифм обеих частей:

    • x = loga(b)

  • Замена переменной: Используется, если уравнение содержит сложные выражения, например 22x - 3·2x + 2 = 0.

Пример решения

Решить уравнение 32x+1 = 27.

Решение:

Переписываем число 27 как степень тройки: 27 = 3³.

Получаем: 32x+1 = 3³.

Приравниваем показатели: 2x + 1 = 3.

Решаем уравнение: 2x = 2, следовательно, x = 1.

Графики логарифмических функций

Применение логарифмов и показательных уравнений

  • Финансы: расчет сложных процентов.
  • Физика: измерение радиоактивного распада.
  • Инженерия: анализ процессов роста и затухания.
  • Информатика: работа с алгоритмами, связанными со сложностью O(log n).
